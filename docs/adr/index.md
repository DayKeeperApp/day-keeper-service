# Architecture Decision Records

This directory captures architecturally significant decisions for the Day Keeper service using the [ADR format](https://adr.github.io/).

## Conventions

- Use the template: [template.md](template.md)
- Name files: `NNNN-short-title.md` (zero-padded, sequential)
- New decisions start as **Proposed**, move to **Accepted** once approved
- Superseded decisions link to their replacement
